# Mantenimientos Annabe

Landing page para servicios de fontanería, electricidad y gas de Alexis Arrocha.

## Estructura del Proyecto

- `index.html`: Archivo principal que contiene la estructura, estilos (Tailwind CDN + Custom CSS) y lógica (Vanilla JS) de la página.

## Tecnologías

- **HTML5**
- **Tailwind CSS** (vía CDN)
- **Vanilla JavaScript**
- **Font Awesome** (Iconos)

## Notas

Este proyecto utiliza Tailwind CSS a través de CDN para prototipado rápido y facilidad de uso sin configuración de build.
